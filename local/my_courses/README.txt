ABOUT

This plugin for Moodle allowing a manager to create and see custom report on defined archtypes


INSTALLATION

Plugin - report_custom_report

Just place the custom_report directory inside your Moodle's report directory.
Install the plugin and browse to:

Site Administration->Plugins Overview -> Custom Reports
